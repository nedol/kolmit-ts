Инструкция по работе с Kolmit

1. Регистрация нового пользователя

  1. Откройте приложение Kolmit.
  2. Заполните форму регистрации:
    • Name: Укажите ваше имя.
    • Password: Придумайте и введите пароль.
    • Confirm Password: Повторите пароль для подтверждения.
  3. Убедитесь, что все поля заполнены корректно, и нажмите кнопку REGISTER.

2. Выбор языка интерфейса

В верхней панели приложения вы можете изменить язык интерфейса. Нажмите на текущий язык (например, "English" или "Українська") и выберите нужный из списка.

3. Выбор темы для практики

  1. После входа в систему откройте главное меню.
  2. Выберите одну из доступных категорий тем:
    • Herhalingsoefeningen
    • Reizen en transport
    • Werk en carrière
    • Media en communicatie
    • Kunst en creativiteit
    • Wetenschap en technologie
  3. Выбранная тема откроет набор упражнений.

4. Работа с упражнениями

  1. Внутри категории откройте тему (например, "Herhalingsoefeningen").
  2. Выберите конкретное упражнение из списка (например, "ABC-lijst normal perfectum").
  3. Отметьте галочкой упражнения, которые хотите выполнить.
  4. Начните выполнение, следуя инструкциям приложения.

5. Прослушивание и взаимодействие

Для начала разговора нажмите на значок телефона в нижнем углу. Приложение предоставит вам аудио или текстовые подсказки для общения.
